# HTTP Assertions

**Note:** Code samples use `rest-assert-assertj`. If you want to use `rest-assert-unit` instead, replace:

```java
assertThat(response).myAssertion(parameters);
```

With:

```java
assertMyAssertion(response, parameters);
```
